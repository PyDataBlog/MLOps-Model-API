<h2>deleteFromStack and deleteAllFromStack</h2>
<HR>

<H3>Syntax:</H3>

<P>
<pre class="syntax">deleteFromStack (<i>thisStack, thisElement</i>);</pre>
</P>

<P>
<pre class="syntax">deleteAllFromStack (<i>thisStack, thisElement</i>);</pre>
</P>

<H3>Purpose:</H3>

<P>
These functions remove instances of <i>thisElement</i> from <i>thisStack</i>. The standard SLUDGE equality test is performed on each element of the stack and the value of <i>thisElement</i> - if they are the same, the element is removed from the stack. The deleteFromStack function will stop after finding and removing one match; deleteAllFromStack will remove all instances of <i>thisElement</i> which are found.
</P>

<H3>Return value:</H3>

<P>
The command returns the number of elements removed from the stack.
</P>
