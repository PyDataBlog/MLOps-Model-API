                    <h2 class="article-post-title">Apuntes para la Instalación de Gentoo Linux en una laptop ASUS</h2>
<p>Pasos para la instalación de Gentoo Linux en la ASUS ROG a mayo de 2017.</p>
                    <p class="article-post-meta">09 mayo 2017</p>
<p>Esta guía parte de un equipo previamente particionado y con una instalación de Gentoo Linux. Aprovechamos que desde una terminal y con el usuario root se puede hacer una nueva instalación, mientras seguimos trabajando en la anterior.</p>
<h3>Preparar la partición</h3>
<p>Formatee la partición, cambie /dev/sdXX por la partición</p>
<div class="highlight"><pre><span></span><code><span class="err"># mkfs.btrfs -f -L Gentoo201701 /dev/sdXX</span>
<span class="err"># mount /mnt/gentoo</span>
</code></pre></div>

<h3>Descarge el stage</h3>
<p>Y descomprima</p>
<div class="highlight"><pre><span></span><code><span class="err"># cd /mnt/gentoo</span>
<span class="err"># tar xvjpf /mnt/blackbox/Compartidos/Software/Gentoo/stage3-amd64-20170105.tar.bz2</span>
</code></pre></div>

<p>Monte</p>
<div class="highlight"><pre><span></span><code><span class="err"># mount -t proc none /mnt/gentoo/proc</span>
<span class="err"># mount --rbind /dev /mnt/gentoo/dev</span>
<span class="err"># mount --rbind /sys /mnt/gentoo/sys</span>
<span class="err"># mkdir /mnt/gentoo/usr/portage</span>
<span class="err"># mount --rbind /usr/portage /mnt/gentoo/usr/portage</span>
</code></pre></div>

<p>De una vez copie make.conf</p>
<div class="highlight"><pre><span></span><code><span class="err"># cp /etc/locale.gen /mnt/gentoo/etc/locale.gen</span>
<span class="err"># cp /etc/portage/make.conf /mnt/gentoo/etc/portage/make.conf</span>
<span class="err"># cp /etc/env.d/02locales /mnt/gentoo/etc/env.d/02locales</span>
<span class="err"># nano /mnt/gentoo/etc/portage/make.conf</span>
</code></pre></div>

<p>Copie los datos de los DNS</p>
<div class="highlight"><pre><span></span><code><span class="err"># cp /etc/resolv.conf /mnt/gentoo/etc/</span>
</code></pre></div>

<h3>Ingrese al nuevo Gentoo Linux</h3>
<p>Haga el <em>chroot</em></p>
<div class="highlight"><pre><span></span><code><span class="c1"># chroot /mnt/gentoo /bin/bash</span>
<span class="c1"># env-update</span>
<span class="c1"># source /etc/profile</span>
<span class="c1"># export PS1=&quot;(new) $PS1&quot;</span>
</code></pre></div>

<p>Configure locale-gen</p>
<div class="highlight"><pre><span></span><code><span class="err"># locale-gen</span>
</code></pre></div>

<p>Verifique y actualice portage</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge -pu portage gcc</span>
</code></pre></div>

<p>Instale el kernel</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge gentoo-sources</span>
<span class="err"># cd /usr/src/linux</span>
<span class="err"># make -j4</span>
<span class="err"># make modules_install</span>
<span class="err"># make install</span>
</code></pre></div>

<p>Establezca la zona horaria.</p>
<div class="highlight"><pre><span></span><code><span class="err"># cd /etc</span>
<span class="err"># cp /usr/share/zoneinfo/Mexico/General localtime</span>
</code></pre></div>

<p>Otras configuraciones</p>
<div class="highlight"><pre><span></span><code><span class="err"># nano /etc/conf.d/keymaps</span>
<span class="err"># nano /etc/conf.d/hostname</span>
<span class="err"># nano /etc/hosts</span>
<span class="err"># nano /etc/fstab</span>
</code></pre></div>

<h3>Con KDE Plasma como interfaz gráfica</h3>
<p>Cambie el profile a Plasma</p>
<div class="highlight"><pre><span></span><code><span class="err"># eselect profile show</span>
<span class="err"># eselect profile list</span>
<span class="err"># eselect profile set default/linux/amd64/13.0/desktop/plasma</span>
<span class="err"># eselect profile show</span>
</code></pre></div>

<p>Instale...</p>
<div class="highlight"><pre><span></span><code><span class="c1"># emerge gentoolkit reiserfsprogs dhcpcd dosfstools app-misc/screen pwgen</span>
</code></pre></div>

<p>Instale daemons</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge ntp syslog-ng vixie-cron</span>
<span class="err"># rc-update add syslog-ng default</span>
<span class="err"># rc-update add vixie-cron default</span>
<span class="err"># rc-update add ntpd default</span>
</code></pre></div>

<p>Instale utilerías del sistema y de sonido</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge pciutils alsa-utils</span>
<span class="err"># rc-update add alsasound boot</span>
</code></pre></div>

<h3>A instalar software</h3>
<p>Instale PostgreSQL</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge postgresql</span>
</code></pre></div>

<p>Instale Apache</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge apache</span>
</code></pre></div>

<p>Instale Ruby</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge ruby nokogiri redcloth dev-ruby/mechanize</span>
</code></pre></div>

<p>Instale utilerías de red y sistemas de archivos</p>
<div class="highlight"><pre><span></span><code><span class="c1"># emerge app-admin/sudo bind-tools traceroute bridge-utils sys-fs/fuse sys-fs/ntfs3g</span>
</code></pre></div>

<p>Instale KDE plasma-desktop, éste pone el entorno básico (es decir, NO instala "todo el combo") para luego ir instalando las aplicaciones a nuestro criterio. Revise dependencias y USE flags..</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge --ask plasma-desktop samba cups xorg-x11</span>
</code></pre></div>

<p>Instale LibreOffice</p>
<div class="highlight"><pre><span></span><code><span class="o">#</span> <span class="n">emerge</span> <span class="n">virtual</span><span class="o">/</span><span class="n">jdk</span>
<span class="o">#</span> <span class="n">emerge</span> <span class="n">libreoffice</span>

<span class="o">#</span> <span class="n">emerge</span> <span class="n">firefox</span>
<span class="o">#</span> <span class="n">emerge</span> <span class="n">gimp</span>
<span class="o">#</span> <span class="n">emerge</span> <span class="n">texlive</span>
<span class="o">#</span> <span class="n">emerge</span> <span class="n">audacity</span>
</code></pre></div>

<p>Instale PHP</p>
<div class="highlight"><pre><span></span><code><span class="o">#</span> <span class="n">emerge</span> <span class="n">php</span>

<span class="o">#</span> <span class="n">emerge</span> <span class="n">networkmanager</span>
</code></pre></div>

<p>Luego KDE Core</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge kdecore-meta</span>
</code></pre></div>

<p>Actualice todo lo demás</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge -uND world</span>
</code></pre></div>

<p>Instale Geany</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge geany geany-themes geany-plugins</span>
<span class="err"># emerge droid freefonts libertine terminus-font ttf-bitstream-vera corefonts</span>
</code></pre></div>

<p>Y de consola...</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge xterm media-sound/moc nmap whois macchanger aircrack-ng</span>
</code></pre></div>

<p>En lugar de kdegraphics-meta</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge spectacle kde-apps/kate okular gwenview</span>
</code></pre></div>

<p>En lugar de kdemultimedia-meta</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge kdenlive kmix mplayer</span>
</code></pre></div>

<p>Instale el GDM ssdm</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge sddm grub gparted</span>
<span class="err"># sddm --example-config &gt; /etc/sddm.conf</span>
<span class="err"># nano -w /etc/sddm.conf</span>
<span class="err"># rc-update add xdm default</span>
</code></pre></div>

<p>Configure /etc/conf.d/xdm con DISPLAYMANAGER="sddm"</p>
<div class="highlight"><pre><span></span><code><span class="err"># nano /etc/conf.d/modules</span>
<span class="err"># nano /etc/conf.d/xdm</span>
<span class="err"># nano /etc/ntp.conf</span>
</code></pre></div>

<p>Configure...</p>
<div class="highlight"><pre><span></span><code><span class="err"># nano /etc/plasma/startup/10-agent-startup.sh</span>
<span class="err"># nano /etc/plasma/shutdown/10-agent-shutdown.sh</span>
</code></pre></div>

<p>Utilerías de KDE</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge kcalc umbrello</span>
<span class="err"># emerge kile</span>
<span class="err"># emerge konqueror</span>
<span class="err"># emerge kde-plasma/kdeplasma-addons</span>
<span class="err"># emerge unarj arj lha lzop rar unrar kde-apps/ark</span>
</code></pre></div>

<p>KDE para una laptop con bluetooth, powerdevil más systemsettings y otras del sistema</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge -pv plasma-meta</span>
</code></pre></div>

<p>Siga instalando...</p>
<div class="highlight"><pre><span></span><code><span class="c1"># emerge gparted</span>
<span class="c1"># emerge gimp inkscape</span>
<span class="c1"># emerge libvirt virt-manager</span>
<span class="c1"># emerge gvim vim</span>
<span class="c1"># emerge audacity</span>
<span class="c1"># emerge k3b vorbis-tools</span>
</code></pre></div>

<p>XFCE4</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge xfce4-meta xfce4-notifyd gtk-engines</span>
</code></pre></div>

<p>Instale componentes de XFCE</p>
<div class="highlight"><pre><span></span><code><span class="c1"># emerge xfce4-volumed xfce4-mixer xfce4-terminal xfce4-taskmanager xfce4-screenshooter</span>
<span class="c1"># emerge xfce4-cpugraph-plugin xfce4-verve-plugin xfce4-mount-plugin xfce4-sensors-plugin xfce4-systemload-plugin</span>
</code></pre></div>

<p>Soporte multimedia y de archivos empacados para Thunar</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge tumbler</span>
<span class="err"># emerge thunar-volman thunar-archive-plugin thunar-media-tags-plugin</span>
</code></pre></div>

<p>Instale iconos</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge gtk-chtheme tango-icon-theme faenza-icon-theme tangerine-icon-theme</span>
</code></pre></div>

<p>Configure arranque de servicios...</p>
<div class="highlight"><pre><span></span><code><span class="err"># rc-update add dbus default</span>
<span class="err"># rc-update add udev default</span>
<span class="err"># rc-update add ntpd default</span>
<span class="err"># rc-update add cupsd default</span>
</code></pre></div>

<p>Zona horaria</p>
<div class="highlight"><pre><span></span><code><span class="err"># cp /usr/share/zoneinfo/Mexico/General /etc/localtime</span>
</code></pre></div>

<p>Contraseña</p>
<div class="highlight"><pre><span></span><code><span class="err"># passwd</span>
</code></pre></div>

<p>Más utilerías</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge tilda mousepad xfwm4-themes xfce4-whiskermenu-plugin alacarte xfce4-clipman-plugin</span>
<span class="err"># emerge pcmanfm gftp</span>
</code></pre></div>

<p>Instale más software</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge tor privoxy arm</span>
<span class="err"># emerge htop net-fs/sshfs</span>
<span class="err"># emerge dia</span>
</code></pre></div>

<p>Kramdawn es inestable, libere dependencias</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge kramdown</span>
</code></pre></div>

<p>Librerías que uso en Python</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge tabulate psycopg</span>
</code></pre></div>

<p>Software necesario para compilar QGIS desde fuentes</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge sci-geosciences/gpsbabel x11-libs/qwtpolar x11-libs/qscintilla dev-db/spatialite dev-python/PyQt4 sci-libs/libspatialindex dev-python/sip dev-python/jinja dev-python/httplib2 dev-python/pytz dev-python/qscintilla-python dev-libs/qjson sci-geosciences/gpsbabel dev-python/python-dateutil</span>
<span class="err"># emerge -uN sci-libs/gdal</span>
</code></pre></div>

<p>Instale R 3.2.2 y R studio</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge rstudio R</span>
</code></pre></div>

<p>Y...</p>
<div class="highlight"><pre><span></span><code><span class="err"># emerge ccache</span>
<span class="err"># emerge sys-process/at</span>
<span class="err"># emerge xarchiver</span>
<span class="err"># emerge app-admin/testdisk</span>
</code></pre></div>

<h3>Configurar el gestor de arranque GRUB 2</h3>
<p>Edite...</p>
<div class="highlight"><pre><span></span><code><span class="err"># nano -w /etc/default/grub</span>
</code></pre></div>

<p>Edite estas líneas, cambie la identificación de la partición swap...</p>
<div class="highlight"><pre><span></span><code><span class="err">GRUB_TIMEOUT=30</span>
<span class="err">GRUB_CMDLINE_LINUX=&quot;resume=UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx kvm-intel.nested=1&quot;</span>
<span class="err">GRUB_GFXMODE=1920x1080</span>
<span class="err">GRUB_GFXPAYLOAD_LINUX=keep</span>
</code></pre></div>

<p>No deje de tener opciones manuales en...</p>
<div class="highlight"><pre><span></span><code><span class="err"># nano -w /etc/grub.d/40_custom</span>
</code></pre></div>

<p>Formatee la partición de 64 MB code EF00 /dev/sdb1</p>
<div class="highlight"><pre><span></span><code><span class="err"># mkfs.vfat -n EFIBOOT /dev/sdb1</span>
</code></pre></div>

<p>Copie la clave UUID de /dev/sdb1</p>
<div class="highlight"><pre><span></span><code><span class="err"># blkid</span>
</code></pre></div>

<p>Y cambie la clave UUID en...</p>
<div class="highlight"><pre><span></span><code><span class="err"># nano -w /etc/fstab</span>
<span class="err"># nano -w /etc/grub.d/40_custom</span>
</code></pre></div>

<p>Con ésto, puede montarla...</p>
<div class="highlight"><pre><span></span><code><span class="err"># mount /boot/efi</span>
</code></pre></div>

<p>Así montada, instale GRUB2...</p>
<div class="highlight"><pre><span></span><code><span class="err"># grub-install --efi-directory=/boot/efi</span>
<span class="err">Instalando para plataforma x86_64-efi.</span>
<span class="err">Instalación terminada. No se notificó ningún error.</span>
</code></pre></div>

<p>Y habilite la configuración...</p>
<div class="highlight"><pre><span></span><code><span class="err"># grub-mkconfig -o /boot/grub/grub.cfg</span>
<span class="err">Generando un fichero de configuración de grub...</span>
