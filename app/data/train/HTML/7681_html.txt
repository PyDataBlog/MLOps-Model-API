<H3><A NAME="SECTION00041100000000000000"></A><A NAME="Algorithm::new"></A>
<BR>
<TT>new()</TT>
</H3>
<I>Arguments:</I> <A HREF="node40.html#xref:Population"><TT>Population</TT></A> <code>=&gt;</code> <SMALL>OBJECT</SMALL>, <SMALL>ATTRIBUTE-HASH</SMALL> (optional) 
<BR><I>Return value:</I> <SMALL>OBJECT</SMALL>
<BR><I>Defined in:</I> PerlGPObject.pm
<BR><I>Mainly called by:</I> perlgp-run.pl
<BR><I>Usually calls:</I> <TT>_init()</TT>
<BR><I>Relevant attributes:</I> all
<BR>
<P>
This is the constructor, but you probably don't need to call it
directly, except in specialist applications.  A Population object must
be given in the argument hash.  Other attributes may also be
specified in the argument hash, but note that the ``usual'' way to
set these attributes is to edit Algorithm.pm in the experiment
directory.
