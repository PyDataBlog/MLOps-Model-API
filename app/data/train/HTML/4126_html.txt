	<h1>Using SHTOOLS with Python</h1>
		
	<p class="nomarginbot">To access the SHTOOLS routines from Python, it is necessary that the <tt>pyshtools</tt> directory be in the Python search path. From within Python, enter</p>

	<blockquote class="small">
		<p class="nomargintop nomarginbot"><tt>import sys<br>
		sys.path.append('/path-to-pyshtools')</tt></p>
   	</blockquote>
	
	<p>where <tt>/path-to-pyshtools</tt> should be <tt>/usr/local/lib/python2.7/site-packages</tt> after a full installation. After setting the path, it is then only necessary to execute the statement</p>
		
		<blockquote class="small">
			<p class="nomargintop nomarginbot"><tt>import pyshtools as shtools </tt></p>
		</blockquote>

	<p>to load the routines and documentation. If you are using <a href="http://ipython.org/">iPython</a>, which adds improved functionality to Python, the available <tt>shtools</tt> routines can be explored by typing</p>
	
	<blockquote class="small">
			<p class="nomargintop nomarginbot"><tt>shtools.[tab]</tt></p>
	</blockquote>
	
	<p>where <tt>[tab]</tt> is the tab key.</p>
	
	<p>To explore the list of predefined constants used by SHTOOLS, enter</p>
	
		<blockquote class="small">
			<p class="nomargintop nomarginbot"><tt>shtools.constant.[tab]</tt></p>
	</blockquote>
	
	<h2>Documentation</h2>
	
	<p>To read the documentation of a routine in iPython, such as <tt>PlmBar</tt>, enter</p>
	
		<blockquote class="small">
			<p class="nomargintop nomarginbot"><tt>shtools.PlmBar?</tt></p>
	</blockquote>
	
	<p>Documentation for the Python functions used in SHTOOLS can also be accessed by their unix man pages, appending <tt>py</tt> to the name and using all lower case letters. As an example, to access the <tt>MakeGridDH</tt> man page, use</p>
	
	<blockquote>
	<p>
	<tt>man pymakegriddh</tt></p>
	</blockquote>

	<p>Alternatively, the man pages can be accessed from the <a href="documentation.html">documentation</a> link on this web site.</p>
