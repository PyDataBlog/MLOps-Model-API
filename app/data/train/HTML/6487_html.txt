<h2>callEvent</h2>
<HR>

<H3>Syntax:</H3>

<P>
<pre class="syntax">callEvent (<i>thisAction, thisObject</i>);</pre>
</P>

<H3>Purpose:</H3>

<P>
Calls the function specified inside the <i>thisObject</i> objectType definition by the line &quot;event <i>thisAction</i>&quot;. Both <i>thisAction</i> and <i>thisObject</i> must be variables of type objectType.
</P>

<H3>Return value:</H3>

<P>
TRUE if a function was called. FALSE if no event was found.
</P>

<H3>Example:</H3>

<P>
<pre>objectType ego () {
}

objectType lookAt () {
}

objectType poster (&quot;poster&quot;) {
   event lookAt {
      say (ego, &quot;I'm looking at the poster.&quot;);
   }
}

sub init () {
   callEvent (poster, lookAt);
      # Returns FALSE

   callEvent (lookAt, poster);
      # Calls function and returns TRUE
}</pre>
</P>
