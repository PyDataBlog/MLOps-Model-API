#jQuery plugin

##How to use
Use it like a normal jQuery plugin <br>
Exemple: ```$("#firstRow .columns").equalise();```

##List of plugins  

```cursorPosition()```  
You can choose the position of the cursor in a input or textarea
  
```positionBottom()```  
Watch element and add position the exact relative position to push the element to the bottom.
  
```equalise()```  
Watch elements and keep the higher one and set it to all
  
```exist()```  
Check if an element exist and return true or false
  
```navSlide({height:{open: int,close: int});```
