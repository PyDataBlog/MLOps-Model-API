# archetypes
Simple maven archetypes for common use

## Installation

 > cd archetypes
 
 > mvn install -Dmaven.test.skip=true

## Configuration

  > mvn archetype:crawl
 
  > cd ~/.m2/repository
 
  > mv archetype-catalog.xml ../
  
### Creating projects(interactive mode)

  > mvn archetype:generate -DarchetypeCatalog=local
