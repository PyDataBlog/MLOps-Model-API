This project contains the 2.x source code of the `fabric8-maven-plugin`

**NOTE** this version of the maven plugin is now considered deprecated.

* the new [3.x or later version of the maven plugin is documented here](https://maven.fabric8.io/)
* the source code for the 3.x maven plugin is in [fabric8io/fabric8-maven-plugin](https://github.com/fabric8io/fabric8-maven-plugin)
