Возвращает правило фильтрации документов.

# Syntax

```js
BaseDataSource.getFilter()
```

## Parameters

Нет

## Returns

Правило фильтрации документов (см. [формат](../BaseDataSource.setFilter/#filter-format))

# Examples

```js
var filter = dataSource.getFilter();
```
