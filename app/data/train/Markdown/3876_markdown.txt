scala_serial
============

Scala interface to RXTX and PureJavaComm

```scala
import ul.commx.{SerialPJC}

//get list of available ports
val ports = SerialPJC.ports

//open port
val port = new SerialPJC("ttyUSB0")

//set parameters
port.baud   = 9600
port.bits   = 8
port.stops  = 2
port.parity = "e"
port.DTR    = true
port.RTS    = false

//get parameters
val ri = port.RI

//get number of available bytes in receive buffer
val avail = port.available

//read received bytes (0 for all)
val data = port.read(0)

//write bytes
port.write(List(0.toByte, 1.toByte, 10.toByte))

//close
port.close
```
