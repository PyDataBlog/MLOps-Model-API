# CPSC3780-Project

CPSC 3780 Data Communications and Networking project

Chat system with client server setup.

Client will be able to send messages to another client connected to a server. 

Message goes from the client to the server, and then goes to the destionation client. 
Check [CPSC3780project.pdf](https://github.com/Clumsy-Coder/CPSC3780-Project/blob/master/CPSC3780project.pdf) for more details.

## Stage 1
- Use basic GUI or terminal chat window to communicate with each other. 
- Have a server setup and be able to communicate with two clients. 

## Stage 2
- Be able to connect to a client who is on a different server. Server to server communication.
- Have a better GUI interface, using JFoenix library.
- Use Gradle, Ant, or Make for building the project and running it on a computer without the need of IDE

## Coding tools and practices.
* Language used: Java SE 1.8.0_102 or higher
* IDE: Intellij-Idea Community Edition [link](https://www.jetbrains.com/idea/)
* GUI: JavaFX (JFoenix library): 
 * [Github](https://github.com/jfoenixadmin/JFoenix) 
 * [Official website](http://www.jfoenix.com/)
* Using GIT branching convention [link](http://nvie.com/posts/a-successful-git-branching-model/)
