# Scala Path
1. Complete [Scala Tutorials Basic Tour](http://scalatutorials.com/tour)
2. Complete [Scala Exercises Std Lib](https://www.scala-exercises.org/std_lib)
3. Read [Scala Style Guide](https://docs.scala-lang.org/style/)
4. Read [Programming in Scala (3rd ed)](https://booksites.artima.com/programming_in_scala_3ed) - chapters 4-21
5. Read [Functional Programming in Scala](https://www.manning.com/books/functional-programming-in-scala) - chapters 1-6
6. Complete [Coursera Functional Programming in Scala Specialization](https://www.coursera.org/specializations/scala)
