# FormComponents

Implement HTML form elements without Javascript and frameworks for keeping native behaviors but customize style.

## Component List:
* Text Input
* Email Input
* Tel Input
* Url Input
* Search
* Radios
* Checkboxes
* Textarea
* Selector
* File Picker
* Numeric Input
* Range Bar
* Switcher
* Buttons

## Class:
* ::Basic
* ::Fantasy (not yet)
