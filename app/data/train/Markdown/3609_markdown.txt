# Requests for new Features
Edit this document for any suggestions on features you would like added.
Feel free to open an issue as well, but this helps us consolidate new features and details. It also can act as a TODO list.

#### The basic template (loose):
Detail Explanations: detail

Please fill out as much information as you find applicable, this may include commands you would like to be implemented.

## Examples

New Feature: dry run

`docker-clean --dry-run`
