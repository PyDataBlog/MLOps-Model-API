Helix is a wrapper library for the Twistage API.

If you are looking to contribute, that is a great place to start.

A lot of the Helix API has gone through design discussions before
implementation. Because of design issues, contributors may want to 
post their feature idea in the GitHub 'Issues' section before implementation.

How to contribute

Fork it
Create your feature branch (git checkout -b my-new-feature)
Create tests for your feature (features without tests will not be included)
Commit your changes (git commit -am 'Add some feature')
Push to the branch (git push origin my-new-feature)
Create new Pull Request
