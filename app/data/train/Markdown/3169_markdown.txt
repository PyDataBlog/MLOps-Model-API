# Premake Extension to support the GCC and Clang toolsets in Visual Studio

Visual Studio support for GCC and Clang toolsets via [vs-tool](https://github.com/juj/vs-tool)

In premake, use the 'toolset' api to choose "gcc" or "clang". Configure the toolchains according to the vs-tool usage instructions.

Supported features:

* MinGW platform
* Clang platform
