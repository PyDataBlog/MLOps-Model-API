\section{JWM SETTINGS MANAGER}

\includegraphics[width=0.7\linewidth]{settings-manager} \\

\paragraph{JWM (Joe's Window Manager) is a lightweight stacking window manager for the X Window System written by Joe Wingbermuehle. (Wikipedia) } 

\paragraph{This section of the ToriOS manual covers the settings manager so that users can be familar with how to configure their system settings (such as mouse, themes) and personalise their new system setup.}

\paragraph{Upon loading settings manager from the menu you will be presented with the following screen shown above.  From here you can access various settings.  For the purpose of this manual we will go from left to right along each row starting at the top left. }

\section{PANEL SETTINGS}

\includegraphics[width=0.7\linewidth]{panel-apps}
\newline 

\paragraph{This section deals with the panel,  the area of the desktop which allows access to : Menus, icons, clock and other widgets}

\paragraph{The first tab allows you to set what appears on the panel and where, the panel is the bar at the top of the screen, you can also configure the menu from here}.
\newpage

\textbf{The appearance settings tab}
\\
\includegraphics[width=0.7\linewidth]{panel-appearance}
\linebreak
\paragraph{Allows you to fine tune settings such as color,  and how Opaque (see through) the panel and menu appear.}

\newpage
\textbf{Finally size and position}

\includegraphics[width=0.7\linewidth]{panel-size-position}
\linebreak
\paragraph{allows you to position the panel on the screen and change how big or small it should appear on the screen. }
