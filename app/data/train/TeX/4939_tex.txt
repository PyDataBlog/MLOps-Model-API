\section{Detection Actions}
\emph{Detection} actions are run periodically during sessions to detect whether
a user is actively using the rig. If there is no use after a permission set
timeout period the session is terminated.

\begin{figure}[htbp]
	\centering
	\begin{minipage}{12cm}
		\textbf{NOTE:} Idle timeout session termination may be disabled in the
		access granting permission. \emph{Detection} actions are still run but have
		no effect on the session.
	\end{minipage}
\end{figure}
