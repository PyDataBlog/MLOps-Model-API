# SQL predicate clauses

SQL where clause support 18 predicates

## Comparisons

`=` equal to
`<>` not equal to
`<` less than
`>` greater than
`<=` less than or equal to
`>=` greater than or equal to

## BETWEEN
To give a range

## IN
To check the value in a list

## LIKE
A text pattern match

## IS NULL
a value expression that is equal to `NULL`
