Bloc Jams Angular
=========

Bloc Jams Angular is [Bloc Jams](https://github.com/ipaulson/bloc-jams) refactored using Angular

- HTML
- CSS
- Javascript
- Angular

& other resources:

- Buzz Library
- Fonts from Google
- Icons from [Ionic](https://ionicframework.com/docs/ionicons/)
- jQuery

Clone It and Run It Yourself
----------------------------

```
git clone https://github.com/ipaulson/bloc-jams-angular
```

Open `index.html` to check it out!
